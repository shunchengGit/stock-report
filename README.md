## 文件命名规则

所有文件遵循统一命名格式：
```
YYYY-MM-DD券商名_股票名（代码）：报告标题.pdf
```

例如：
- `2026-04-27-0700.HK-HSBC Global Investme-Tencent Holdings (700 HK) Buy ...`
- `20260430-华泰证券-微软-MSFT.US-看好Azure业绩修复及Copilot放量.pdf`

## 时间分布

- **最新文件**：2026年5月（剂泰科技招股书、三星电子、策略研究）
- **最旧文件**：2025年11月（微软）
- **主要覆盖**：2025年11月 - 2026年5月（近半年）

## 使用建议

1. **按标的分析**：进入对应目录查看该公司的所有研报
2. **时间排序**：文件名已按日期排序，最新研报在前
3. **券商视角**：不同券商对同一公司可能有不同观点，建议对比阅读
4. **行业交叉**：部分研报包含行业分析，可参考相关行业观点

## 更新维护

1. 新下载的研报请按命名规则重命名
2. 放入对应的标的目录（无对应目录则新建）
3. 中小券商研报不纳入本库，仅保留头部券商与外资大行
4. 更新本README中的统计信息
